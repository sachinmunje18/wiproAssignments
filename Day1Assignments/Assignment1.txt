1. IP Address

Definition: Unique identifier for devices on a network.
Example: 192.168.1.1
---------------------------------------------------------------------------
2. Subnet Mask

Definition: Divides IP address into network and host parts.
Example: 255.255.255.0
---------------------------------------------------------------------------
3.MAC Address

Definition: Unique hardware address for network devices.
Example: 00:1A:2B:3C:4D:5E
---------------------------------------------------------------------------
4.DNS

Definition: Translates domain names to IP addresses.
Example: www.example.com -> 93.184.216.34
---------------------------------------------------------------------------
5.DHCP

Definition: Automatically assigns IP addresses on a network.
Example: Laptop getting IP from Wi-Fi router.
---------------------------------------------------------------------------
6.Router

Definition: Forwards data between networks.
Example: Home router to ISP.
---------------------------------------------------------------------------
7.Switch

Definition: Connects devices within a single network.
Example: Ethernet switch in an office.
---------------------------------------------------------------------------
8.Firewall

Definition: Controls incoming and outgoing network traffic.
Example: Corporate firewall.
---------------------------------------------------------------------------
9.VPN

Definition: Encrypts internet connection for secure access.
Example: Secure connection to work network.
---------------------------------------------------------------------------
10.Bandwidth

Definition: Maximum data transfer rate.
Example: 100 Mbps internet connection.
---------------------------------------------------------------------------
11.Latency

Definition: Delay in data transfer.
Example: 20 ms ping time.
---------------------------------------------------------------------------
12.Packet

Definition: Small segment of data transmitted over a network.
Example: Portion of an email.
---------------------------------------------------------------------------
13.Protocol

Definition: Set of rules for data exchange.
Example: HTTP for web pages.
---------------------------------------------------------------------------
14.NAT

Definition: Translates private IP addresses to a public IP.
Example: Home router allowing multiple devices to share one IP.
---------------------------------------------------------------------------
15.SSID

Definition: Identifier for a wireless network.
Example: "HomeNetwork"
---------------------------------------------------------------------------
16.VoIP

Definition: Voice communication over IP networks.
Example: Skype calls.
---------------------------------------------------------------------------
17.Ethernet

Definition: Wired networking technology.
Example: Ethernet cable connection.
---------------------------------------------------------------------------
18.QoS

Definition: Manages network traffic to reduce delays.
Example: Prioritizing video calls.
---------------------------------------------------------------------------
19.Topology

Definition: Network layout.
Example: Star topology.
